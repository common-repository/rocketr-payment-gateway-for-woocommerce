== Description ==

This plugin allows you to easily integrate Rocketr.net with your existing woocommerce store. With Rocketr, you can easily accept a multitude of payment methods such as Bitcoin, Bitcoin Cash, Ethereum, PayPal, Stripe (Credit cards), Perfect Money and more.


== Installation ==

Installing the plugin is very simple. Just follow these steps:

1. Upload the plugin by going to your Wordpress Dashboard > Plugins > Add New > Upload Plugin > Upload the `rocketr-woocommerce.zip` file.
2. In the WooCommerce Settings page, click on Checkout, and then click on Rocketr.net.
3. In the settings page, type in your Rocketr API credentials and your Rocketr IPN Secret. You can find these at https://rocketr.net/merchants/api.
4. Click on "Save changes".
